IF USER SKIP,USER WON'T BE ABLE TO PEFORM SOME OPERATIONS,THUS CAN'T VISIT SOME PAGES,AND USER WILL BE REDIRECTED TO SIGNUP/LOGIN PAGE.

- **CAN'T MAKE A COMMENT ON PRODUCTS.**
- **CAN'T PERFORM BUY NOW OPEARTIONS.**
- **CAN'T CONTACT US.**
- **CAN'T CHECK OUT.**
- **CAN'T HAVE PROFILE.**
- **CAN'T HAVE ORDER HISTORIES.**

- **RATING TO AVAILABILTY,PRICE TO INFORMATION,IMAGES AND DETAILS**

LOGICAL FUNCTIONS HAVE BEEN USED TO SHOW YOU PERFECT RESULTS,TO CLEAR FILTERS JUST CLICK ON RED BUTTON LABELLED AS 'CLEAR FILTERS'.APPLYING FILTERS WILL CHANGE YOUR PRODUCTS AND ABOVE IT ALSO CHANGE DEATAIL AND BANNER.

# IF YOU ARE NOT LOGIN,STILL YOU CAN ADD YOUR FAVOURITE PRODUCTS TO WISHLIST.WHERE YOU WILL BE ABLE TO SEE NAME,PRICE,AVAILABLITY AND CAN REMOVE THEM TOO,EITHER ONE OR ALL.BUT ONCE YOU ADD IT,YOU CAN'T ADD AGAIN.AS IT IS ALREADY ADDED.BUT IF USER CLICKS ON HEART IMAGE AGAIN,THE ITEM WILL BE REMOVED FROM WISHLIST,THE RED OUTLINE HJEART MEANS THAT THIS ITEM IS NOT ADDED TO WISHLIST,WHILE RED COLORFUL HEART WILL GIVE USER INDICATION THAT THIS ITEM IS ALREADY ADDED TO WISHLIST.
